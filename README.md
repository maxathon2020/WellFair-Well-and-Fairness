# Run Locally
----
-   Clone our repo 
-   Install the required packages via yarn 
-   Start the app with yarn
-   Application should start at port 3000
```
git clone https://github.com/maxathon2020/WellFair-Well-and-Fairness
yarn
yarn start
```
